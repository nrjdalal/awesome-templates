| URL Shortener Example (contributor) | #239 | Adds `examples/url_shortener/` — a CRUD `link` domain + a Taskiq `cleanup_expired_links_task` that shares the same `LinkService` as the HTTP router (mirrors `examples/todo/` layout + the `src/user` worker pattern). | Core not-found 404 fix | #246 | `Database.session()` caught every in-block exception — including a domain `BaseCustomException` (e.g. the 404 a repository raises on a missing row) — and re-wrapped it as `DatabaseException(500, DB_INTERNAL_ERROR)`, so every domain's "get by id → missing row" returned 500 instead of 404 (and in dev leaked the internal message into `errorDetails.original_error`). Adds `except BaseCustomException: raise` ahead of the catch-all so typed domain/infra exceptions (and already-typed `DatabaseException`) propagate unchanged; only genuine driver errors become a 500. Mirrors `src/classification`'s Protocol + Adapter + Selector + Stub wiring → degrades to a deterministic `StubChatbot` when no LLM provider is configured. Deliberately minimal (no endpoint auth; runtime guardrails deferred to #250; `tokens_used` not routed through `ai_usage` #75) with production caveats documented in the example README. `examples/`-only — no `src/` runtime change.
