(`/console/docs/runbooks/incident-response`) returns a 404 for anyone who is not
an internal/admin user, exactly like the console root.

## Checklist

1. Acknowledge the alert.
2. Open an incident channel.
3. Identify the blast radius and mitigate.
4. Write up a postmortem.
